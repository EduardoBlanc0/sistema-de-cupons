Se preferir testar localmente, siga os passos abaixo:

Pré-requisitos:
* Possuir o Java JDK instalado (versão 11 ou superior).

### Passo a Passo
1. **Clone o repositório:**
gh repo clone EduardoBlanc0/sistema-de-cupons

3. Entre na pasta do projeto:
cd SistemaCupons

4. Compile e execute o aplicativo com um único comando:
javac SistemaCupons.java e java SistemaCupons
